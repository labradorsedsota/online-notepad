## 2. 产品目标与背景

### 背景

本应用是 VLA 测试应用池（App Pool）的第一个应用，属于 A 类纯工具型应用。用于 VLA 模型测试数据采集，同时为用户提供一个实用的在线记事工具。

### 产品目标

- 提供一个打开即用的浏览器端记事本，无需注册、无需后端
- 支持多种格式导出（TXT / Markdown / HTML）
- 纯前端实现，GitHub Pages 部署
- 自然覆盖 11 个 UI 组件，为 VLA 测试提供数据采集基础

### 成功标准

- 用户打开页面即可开始记录，零学习成本
- 笔记可保存到浏览器本地存储，刷新不丢失
- 支持 3 种格式一键导出下载
- 暗色模式与字体切换正常工作

## 4. 功能需求

### F-01: 笔记标题输入

**描述：** 页面顶部提供标题输入框，用户可输入笔记标题。

**使用组件：** Input

**AC：**

- **AC-01-1:** Given 用户打开页面，When 页面加载完成，Then 标题输入框显示，placeholder 为"输入笔记标题…"
- **AC-01-2:** Given 用户在标题框中输入文字，When 输入内容，Then 标题实时显示在输入框中
- **AC-01-3:** Given 用户已保存过笔记，When 刷新页面，Then 标题框恢复上次保存的标题内容

### F-02: 笔记内容编辑

**描述：** 主编辑区为大面积文本输入区域（textarea），用于编写笔记正文。内容区包裹在卡片容器中。

**使用组件：** Input（textarea 模式）、Card

**AC：**

- **AC-02-1:** Given 用户打开页面，When 页面加载完成，Then 显示卡片包裹的文本编辑区域，placeholder 为"开始记录…"
- **AC-02-2:** Given 用户在编辑区输入文字，When 输入内容，Then 文字实时显示，编辑区自动扩展高度（不出现内部滚动条，最小高度 300px）
- **AC-02-3:** Given 用户已保存过笔记，When 刷新页面，Then 编辑区恢复上次保存的内容
- **AC-02-4:** Given 编辑区为空，When 用户未输入任何内容，Then 显示 placeholder 文字

### F-03: 保存笔记

**描述：** 点击保存按钮，将标题和内容保存到浏览器 localStorage。

**使用组件：** Button、Message

**AC：**

- **AC-03-1:** Given 用户已输入标题和/或内容，When 点击"保存"按钮，Then 数据保存到 localStorage，显示成功提示消息"保存成功"
- **AC-03-2:** Given 标题和内容均为空，When 点击"保存"按钮，Then 显示警告提示"内容为空，无法保存"
- **AC-03-3:** Given 用户已保存笔记，When 关闭浏览器后重新打开页面，Then 标题和内容恢复

### F-04: 导出笔记

**描述：** 用户选择导出格式后点击导出，弹出确认对话框，确认后下载文件。

**使用组件：** Button、Radio、Dialog、Message

**AC：**

- **AC-04-1:** Given 用户已输入内容，When 点击"导出"按钮，Then 弹出导出确认对话框
- **AC-04-2:** Given 导出对话框已打开，When 对话框显示，Then 显示格式选择（Radio：TXT / Markdown / HTML），默认选中 TXT
- **AC-04-3:** Given 用户选择了 TXT 格式并确认，When 点击"确认导出"，Then 下载 `.txt` 文件，文件名为标题（无标题则为"未命名笔记"），文件内容为纯文本
- **AC-04-4:** Given 用户选择了 Markdown 格式并确认，When 点击"确认导出"，Then 下载 `.md` 文件，标题作为一级标题（`# 标题`），正文在标题下方
- **AC-04-5:** Given 用户选择了 HTML 格式并确认，When 点击"确认导出"，Then 下载 `.html` 文件，含完整 HTML 结构（`<html><head><title>标题</title></head><body>...`），正文段落用 `<p>` 包裹（按换行分段）
- **AC-04-6:** Given 导出对话框已打开，When 点击"取消"或对话框外部，Then 关闭对话框，不执行导出
- **AC-04-7:** Given 标题和内容均为空，When 点击"导出"按钮，Then 显示警告提示"内容为空，无法导出"，不弹出对话框

### F-05: 清空笔记

**描述：** 清空标题和内容，重新开始。

**使用组件：** Button、Alert

**AC：**

- **AC-05-1:** Given 用户已输入内容，When 点击"清空"按钮，Then 显示确认提示"确定要清空所有内容吗？此操作不可撤销"
- **AC-05-2:** Given 确认提示已显示，When 用户确认清空，Then 标题和内容清空，localStorage 中的保存数据也清除
- **AC-05-3:** Given 确认提示已显示，When 用户取消，Then 不做任何操作
- **AC-05-4:** Given 标题和内容均为空，When 点击"清空"按钮，Then 显示信息提示"已经是空白状态"

### F-06: 暗色模式切换

**描述：** 提供暗色/亮色模式切换开关。

**使用组件：** Switch

**AC：**

- **AC-06-1:** Given 页面默认为亮色模式，When 用户切换 Switch 为开启状态，Then 页面切换为暗色模式（背景深色、文字浅色）
- **AC-06-2:** Given 页面为暗色模式，When 用户切换 Switch 为关闭状态，Then 页面切换回亮色模式
- **AC-06-3:** Given 用户已切换模式，When 刷新页面，Then 保持上次选择的模式（存储到 localStorage）

### F-07: 字体选择

**描述：** 下拉选择编辑区的显示字体。

**使用组件：** Select

**AC：**

- **AC-07-1:** Given 页面加载完成，When 查看字体选择器，Then 显示字体下拉菜单，至少包含：默认、等宽（Monospace）、衬线（Serif）、无衬线（Sans-serif）
- **AC-07-2:** Given 用户选择了一种字体，When 选择完成，Then 编辑区文字立即切换为对应字体
- **AC-07-3:** Given 用户已选择字体，When 刷新页面，Then 保持上次选择的字体（存储到 localStorage）

### F-08: 按钮提示

**描述：** 鼠标悬停在操作按钮上时，显示功能说明。

**使用组件：** Tooltip

**AC：**

- **AC-08-1:** Given 用户将鼠标悬停在"保存"按钮上，When 悬停超过 300ms，Then 显示 Tooltip "保存笔记到浏览器"
- **AC-08-2:** Given 用户将鼠标悬停在"导出"按钮上，When 悬停超过 300ms，Then 显示 Tooltip "导出为文件"
- **AC-08-3:** Given 用户将鼠标悬停在"清空"按钮上，When 悬停超过 300ms，Then 显示 Tooltip "清空所有内容"
- **AC-08-4:** Given Tooltip 已显示，When 鼠标移出按钮区域，Then Tooltip 消失

### F-09: 底部链接

**描述：** 页面底部提供关于和帮助链接。

**使用组件：** Link

**AC：**

- **AC-09-1:** Given 页面加载完成，When 查看页面底部，Then 显示"关于"和"帮助"两个链接
- **AC-09-2:** Given 用户点击"关于"链接，When 点击，Then 显示关于信息（可用 Dialog 或跳转到 repo 页面）
- **AC-09-3:** Given 用户点击"帮助"链接，When 点击，Then 显示使用说明信息

### F-10: 未保存提醒

**描述：** 当用户有未保存的修改时，显示持久化的提醒条。

**使用组件：** Alert

**AC：**

- **AC-10-1:** Given 用户修改了标题或内容（相对于上次保存），When 内容发生变化，Then 页面顶部显示 Alert 提醒"您有未保存的更改"
- **AC-10-2:** Given 未保存提醒已显示，When 用户点击保存，Then Alert 消失
- **AC-10-3:** Given 内容未修改（与上次保存一致），When 页面无变动，Then 不显示 Alert

## 5. 业务规则与边界条件

| 编号 | 规则 | 说明 |
|------|------|------|
| BR-01 | 标题长度限制 | 最多 100 个字符，超出截断 |
| BR-02 | 内容无长度限制 | 但 localStorage 有 5MB 上限，超出时提示存储已满 |
| BR-03 | 文件名生成 | 导出文件名 = 标题内容（去除特殊字符），无标题时为"未命名笔记" |
| BR-04 | 导出文件名特殊字符 | 文件名中的 `/\:*?"<>\|` 替换为 `_` |
| BR-05 | 默认状态 | 首次打开：亮色模式、默认字体、空白内容 |
| BR-06 | localStorage Key | 使用 `vla-ntp-` 前缀，避免与其他应用冲突 |

## 6. 异常与错误设计

| 场景 | 处理方式 |
|------|---------|
| localStorage 已满 | 保存时 catch 异常，显示 Message 错误提示"存储空间已满，请导出后清空" |
| 浏览器不支持 localStorage | 页面顶部 Alert 警告"当前浏览器不支持本地存储，笔记将无法保存" |
| 导出空内容 | 阻止导出，显示 Message 警告提示 |
| 文件下载失败 | 显示 Message 错误提示"导出失败，请重试" |

## 11. 风险与决策记录

### 风险

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| localStorage 5MB 限制 | 大量文字可能超限 | 保存时 catch 异常并提示用户导出 |
| 单 HTML 文件体积 | 代码量增长可能影响加载 | 保持精简，CSS/JS 内联 |
| CDN 依赖可用性 | 若使用外部 CDN，断网时影响 | 选择高可用 CDN 或考虑全内联 |

### 决策记录

| 编号 | 日期 | 决策 | 理由 |
|------|------|------|------|
| D-01 | 2026-04-13 | 采用单 HTML 文件方案 | 与 OMS 项目保持一致，部署简单 |
